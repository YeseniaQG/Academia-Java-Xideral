# Xideral
En este repositorio se encontraran los ejercicios de todas las semanas de la Academia de Java Xideral 
En el paquete com.semana vienen los ejercicios del 1 al 5 
En el paquete com.arraque viene el ejercicio 6 
Las carpetas de las semanas seran nombradas "Semana 1, Seemana2..."
-Cada una de las carpetas contiene:
                                    - Carpeta de Ejercicios
                                    - Carpeta del proyecto (Este puede contener el proyecto base y el proyecto realizado)
